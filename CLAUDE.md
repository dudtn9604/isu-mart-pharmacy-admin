# Claude 프로젝트 설정

## 필수 규칙

1. **세션 시작 시 반드시 `SESSION_LOG.md`를 먼저 읽고, 이전 맥락을 완전히 파악한 상태에서 대화를 시작할 것.**
2. 작업 중 주요 결정사항, 변경사항, 진행 내용을 `SESSION_LOG.md`에 지속적으로 반영할 것.
3. 세션 종료 시 (사용자가 마무리하거나 대화가 끝날 때) 반드시 `SESSION_LOG.md`를 최신 상태로 업데이트할 것.
4. 사용자에게 "로그를 읽어라"라고 지시받지 않아도 **항상 자동으로** 위 과정을 수행할 것.

## 대화 스타일

- 한국어로 대화
- 이전 세션의 맥락을 자연스럽게 이어갈 것
- 축적된 결정사항과 히스토리를 기반으로 일관성 있게 작업할 것
